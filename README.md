# UrbanShare 🛠️

UrbanShare is a modern peer-to-peer (P2P) marketplace platform built for sharing tools with your neighbors. Stop buying things you'll only use once—list your idle tools and earn extra income, or borrow what you need locally.

---

## 🚀 Key Features

* **P2P Tool Marketplace:** List, browse, and rent various tools.
* **Booking System:** Request rentals, manage dates, and approve requests.
* **Real-time Notifications:** Stay updated with your rental status via in-app notifications.
* **User Reviews:** Build trust in the community with rating and feedback systems.
* **Dashboard Management:** Track your listed tools and current bookings in one place.
